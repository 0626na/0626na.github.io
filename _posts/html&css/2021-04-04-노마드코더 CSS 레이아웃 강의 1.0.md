# 노마드코더 CSS 레이아웃 마스터강의 정리노트입니다.

## 1.0 Life Before Flexbox
HTML에서 block 속성은 옆에 다른 element(요소)가 오는것이 불가능하다. 이러한 block 속성을 유지하면서 요소옆에 다른 요소가 올수 있는 Inline-block 속성이 만들어졌다. 하지만 이 Inline-block 속성은 레이아웃을 지정하는 것이 매우 불편하며 개발자가 전부 수동으로 코드를 변경해야한다. 레이아웃을 화면에 맞게 설정해도 디스플레이가 바뀌면 (예를 들면 컴퓨터 모니터에서 스마트폰), 레이아웃이 전부 깨져버렸다. 이렇듯, 디스플레이의 크기에 맞게 동적으로 레이아웃을 설정해주는 flex-box 속성이 생겨났다.

- block 속성의 요소는 동일 선상에 있을수 없다.
- Inline-block 속성은 block 속성의 요소를 동일 선상에 있을수 있게한다. 하지만 레이아웃 조작이 매우 불편하다.
- 레이아웃을 동적으로 자동 지정해주는 flexbox 속성이 나타나게 되었다.
