# Vue.js??
뷰는 웹 페이지 화면을 개발하는 프론트엔드 프레임워크입니다. 간단히 말하면 화면단 라이브러리이자 프레임워크입니다.
html/css와 js만 알고있으면 쉽게 배울수있게 만들어져 있습니다.

뷰는 코어라이브러리에서 화면단 데이터 표현에 관한 기능을 중점적으로 지원하지만 프레임워크의 기능인 라우터, 상태관리, 테스팅 등을 쉽게 결합할수있는
형태로도 제공합니다. 그렇기에 라이브러리뿐만 아닌 프레임워크의 역할도 할수 있다라는 의미입니다. 그렇기에 뷰를 점진적 프레임워크(progressive framework)라고 부릅니다.

![vue framework](https://blog.kakaocdn.net/dn/uAEYK/btqEBQq9oDo/IDRGJTQeMal5b7A0Y70gn0/img.png)
## 뷰의 장점
1. 뷰는 배우기가 쉽다. html/css와 js만 알면 하루만에 배울수있고, 현직 개발자는 반나절이면 가능하다.
2. 리액트, 앵귤러에 비하면 성능이 우수하고 빠르다.
3. 리액트, 앵귤러의 장점을 가지고있다. 앵귤러의 데이터바인딩 특성과 리액트의 가상돔 기반 렌더링 특징을 전부 가지고 있다.
