# 노마드코더 CSS 레이아웃 강의 정리노트 입니다.
## 1.3 Column and Row
flexbox에서 flex direction은 가로(row)가 기본이다. 하지만 flex direction을 세로(column)으로 바꾸면 main axis가 세로, cross axis가 가로가 된다.
flexbox를 다룰때는 이 main axis와 cross axis를 잘 확인해야 한다.

- flex-direction:row

  main-axis: row
  
  cross-axis: column
- flex-direction: column

  main-axis: column
  
  cross-axis: row

프로퍼티는 방향에 상관없이 동일하다.
- main-axis: justify-content
- cross-axis: align-items
