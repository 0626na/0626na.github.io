자바스크립트에서는 배열을 사용하기 위해서 Array라는 내장객체를 사용합니다. 배열을 인스턴스로 이용하는데에는 몇가지 방법이 있습니다.

```jsx
const myArray = new Array(); //배열 인스턴스 생성 법

const numbers=['one','two','three']; // 리터럴 사용법
const numbers2=new Array('one','two','three'); //Array 객체를 이용한 사용법
//초기값이 없을 경우에는 그냥 빈칸으로 두면 된다.
```

## Array 객체의 함수

array 객체에는 배열을 다룰수 있는 여러 함수를 제공합니다.

### concat() 함수

기존의 배열에 또다른 배열이나 값을 합쳐서 새로운 배열을 만드는 함수입니다.

```jsx
let nums = ['1','2','3'];
let chars = ['a', 'b', 'c'];

let result = nums.concat(chars);
// ['1','2','3','a', 'b', 'c'] 새로운 배열이 생성
// 새로운 배열을 생성하기에 기존의 배열들에는 영향을 주지 않는다.
```

### join() 함수

배열 요소를 연결하는 함수입니다.  요소를 연결할때 요소 사이에 넣을 구분 기호를 지정할수 있습니다. 지정하지 않으면 기본쉼표로 지정됩니다.

```jsx
let nums = ['1','2','3'];
nums.join();
// '1,2,3' 
nums.join('-');
// '1-2-3'
// 요소를 연결하여 연결된 값을 반환한다.
```

### push()함수, unshift() 함수

기존의 배열에 새로운 값을 추가하는 함수들입니다. 배열 맨끝에 추가할때는 push() 함수, 맨앞에 추가할때는 unshift() 함수를 사용합니다. 

```jsx
let nums = ['1','2','3'];
nums.push('4', '5'); //추가하고 싶은 값들을 입력
// 기존의 nums의 맨뒤에 4와 5를 추가 ['1', '2', '3', '4', '5']

nums.unshift('0');
// nums의 맨앞에 0이 추가 ['0', '1', '2', '3', '4', '5']
```

### pop() 함수, shift() 함수

array 객체에서 요소들을 추출합니다. 추출한 값들은 배열에서 삭제 됩니다. 맨앞에 요소를 추출할때는 shift() 함수, 맨 뒤의 요소를 추출할떄는 pop() 함수를 사용합니다.

```jsx
let study = ['html', 'css', 'javascript'];
study.pop();
// ['html', 'css'] 맨뒤에 요소가 추출되어 사라짐

study.shift();
//['css'] 맨앞의 요소가 사라짐
```

### splice() 함수

배열 중간에 요소를 추가하거나 삭제하는 함수입니다.  괄호안의 인수에 따라서 일정구간의 요소를 삭제하거나 새로 추가하는 함수입니다.

```jsx
let numbers = [0,1,2,3,4,5];
numbers.splice(2); // 인수가 하나인 경우는 배열의 위치를 나타낸다.
									 // 3번째 요소부터 모두 삭제한다.
// [0,1]

numbers.splice(2,1);
// 인수가 두개인 경우는 첫번째 인수는 인덱스 값이고 두번째 인수는 삭제할 요소의 갯수이다.
// 결과는 [0,1,3,4,5]

numbers.splice(2,1,'js');
//인수가 3개인 경우는 첫번째는 인덱스 두번째는 삭제할 갯수 세번째는 삭제한 요소를
//대체할 새로운 값입니다.
// 결과는 [0,1,'js',3,4,5]

numbers.splice(2,0,'js');
//기존의 배열값을 삭제하지 않고 추가하려면 삭제할 갯수를 0으로 지정하면 됩니다.
// 결과는 [0,1,'js',2,3,4,5]

numbers.splice(2,0,100,200,300);
//여러 값을 추가하려는 경우는 세번째 인수에 복수의 값을 입력하면 됩니다.
//[0,1,100,200,300,2,3,4,5]
```

### slice() 함수

배열에서 여러개의 값을 꺼내어 새로운 배열을 만드는 함수입니다.  slice() 함수에 시작인덱스와 끝 인덱스를 지정하여 값을 가져옵니다.

```jsx
let numbers = [0,1,2,3,4,5];
numbers.slice(2) //인수를 하나만 입력하면 시작인덱스가 되어 배열 끝까지 모든 값을 가져옵니다.
//[2,3,4,5]; 

numbers.slice(2,3); //2번째 인수에서 3번째 인수까지의 값을 가져와서 배열을 생성
//[2,3];

//이때 slice() 함수는 값을 추출하여 새로운 배열을 만들기에 기존의 배열에 영향을 주지 않습니다.
```

이외에도 Array 객체는 다양한 함수를 제공합니다. 모든 함수를 살펴보려면 모질라 재단의 공식 홈페이지에서 확인이 가능합니다.
